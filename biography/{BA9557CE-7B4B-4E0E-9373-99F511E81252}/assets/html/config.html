通知标签
如果选中，则当前音轨的已启用标签将通知其它蜘蛛猴面板 (Spider Monkey Panels)。这包括电台的艺术家数据。在音轨变化和收到新的或更新的数据时，会发出通知。 当前音轨是正在播放音轨或选定音轨，具体取决于面板显示模式。使用默认标签名称而不是任何自定义名称，因此第三方脚本知道会接收什么。 此外，当前音轨更改时，第三方脚本可以使用 window.NotifyOthers('bio_syncTags', focus)，其中 focus 为 false（首选正在播放项）或 true（跟随所选音轨）。这会覆盖简介(biography)标签设置，以便简介(biography)始终通知标签，通知标签包括一组核心标签，并且通知标签包括正确的选择模式。 在接收面板中使用 on_notify_data(name, info) 来处理通知，其中： name is 'biographyTags' info is an object: { artist: 'ARTIST NAME', album: 'ALBUM NAME', handle: handle of current track, selectionMode: 简介选择模式 - 字符串是 '首选正在播放项' 或 '跟随所选音轨（播放列表）' tags: { {name: value}, {name: value}... } } 第三方脚本可以通过使用 window.NotifyOthers('bio_followSelectedTrack', boolean) 来设置简介选择模式*，其中 boolean 是 true [跟随所选音轨（播放列表）] 或 false [首选正在播放项]。*如果有启用的歌词或正在播放来源，则不适用，因为面板必须处于首选正在播放模式。
